# Diagrammes BPMN - Processus Métier

Ce dossier contient les diagrammes BPMN (Business Process Model and Notation) représentant les processus métier du système.

## Processus Principaux

### 1. Processus Voyage Scolaire
- **Fichier** : `01_processus_voyage_scolaire.bpmn`
- **Description** : Processus complet de gestion d'un voyage scolaire depuis la demande jusqu'à la facturation

### 2. Processus Voyage Linguistique
- **Fichier** : `02_processus_voyage_linguistique.bpmn`
- **Description** : Processus d'inscription et de paiement pour les voyages linguistiques

### 3. Processus Génération Devis
- **Fichier** : `03_processus_generation_devis.bpmn`
- **Description** : Processus de génération et validation d'un devis

### 4. Processus Facturation
- **Fichier** : `04_processus_facturation.bpmn`
- **Description** : Processus de génération et validation d'une facture

## Outils Recommandés

- **Camunda Modeler** : https://camunda.com/download/modeler/
- **bpmn.io** : https://bpmn.io/toolkit/bpmn-js/
- **Visual Paradigm** : Outil professionnel de modélisation BPMN

## Format

Les fichiers BPMN utilisent le format XML standard BPMN 2.0.
