# Liste des Acronymes

## A

**API** : Application Programming Interface (Interface de programmation applicative)

**AJAX** : Asynchronous JavaScript and XML

## B

**BPMN** : Business Process Model and Notation

## C

**CORS** : Cross-Origin Resource Sharing

**CRUD** : Create, Read, Update, Delete

**CRM** : Customer Relationship Management

## D

**DMZ** : Demilitarized Zone (zone démilitarisée réseau)

## E

**ERP** : Enterprise Resource Planning (progiciel de gestion intégré)

## F

**2FA** : Two-Factor Authentication (authentification à deux facteurs)

**FastAPI** : Framework Python pour APIs

## G

**GDPR** : General Data Protection Regulation (RGPD en français)

## H

**HTTPS** : HyperText Transfer Protocol Secure

**HTTP** : HyperText Transfer Protocol

## J

**JSON** : JavaScript Object Notation

**JWT** : JSON Web Token

## M

**MCD** : Modèle Conceptuel de Données (Merise)

**MLD** : Modèle Logique de Données (Merise)

**MPD** : Modèle Physique de Données (Merise)

**MySQL** : Système de gestion de base de données

## O

**ORM** : Object-Relational Mapping

## P

**PCI-DSS** : Payment Card Industry Data Security Standard

**PDF** : Portable Document Format

**PEP 8** : Python Enhancement Proposal 8 (standards Python)

## R

**REST** : Representational State Transfer

**RGPD** : Règlement Général sur la Protection des Données

**ROI** : Return on Investment (retour sur investissement)

**RPC** : Remote Procedure Call

**RTO** : Recovery Time Objective

**RPO** : Recovery Point Objective

## S

**SLA** : Service Level Agreement

**SMTP** : Simple Mail Transfer Protocol

**SPA** : Single Page Application

**SQL** : Structured Query Language

## T

**TLS** : Transport Layer Security

**TOTP** : Time-based One-Time Password

## U

**UML** : Unified Modeling Language

**URL** : Uniform Resource Locator

## V

**VAT** : Value Added Tax (TVA en français)

## W

**WCAG** : Web Content Accessibility Guidelines

**XML** : eXtensible Markup Language

**XML-RPC** : XML Remote Procedure Call
