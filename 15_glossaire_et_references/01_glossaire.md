# Glossaire

## A

**Activité** : Événement ou visite organisée dans le cadre d'un voyage (musée, excursion, etc.)

**ActivityTemplate** : Modèle réutilisable d'activité avec prix et durée standards

**Admin** : Utilisateur avec tous les droits d'administration du système

**Alembic** : Outil de migration de base de données pour SQLAlchemy

**API** : Interface de programmation applicative (Application Programming Interface)

## B

**Backend** : Partie serveur de l'application (FastAPI)

**Booking** : Réservation d'un voyage linguistique par un participant

**BPMN** : Business Process Model and Notation (notation de processus métier)

## C

**Commercial** : Utilisateur responsable de la gestion commerciale (voyages, devis, factures)

**Comptable** : Utilisateur responsable de la validation des factures et de la comptabilité

## D

**Destination** : Lieu de voyage (ville, pays) avec prix de base

**Devis** : Document commercial proposant un prix pour un voyage

**DRAFT** : Statut initial (brouillon) d'un voyage ou devis

## E

**ERP** : Enterprise Resource Planning (progiciel de gestion intégré, ex: Odoo)

## F

**Facture** : Document comptable généré depuis un devis validé

**Factur-X** : Format XML standardisé pour la facturation électronique (conformité 2027)

**FastAPI** : Framework Python moderne pour créer des APIs REST

**Frontend** : Partie client de l'application (Vue.js)

## G

**Guest** : Invité participant à un voyage linguistique

## H

**Hébergement** : Logement pendant un voyage (prix par personne et par nuit)

## I

**Invoice** : Facture (en anglais)

**InnoDB** : Moteur de stockage MySQL recommandé pour les transactions

## J

**JWT** : JSON Web Token, méthode d'authentification sécurisée

## L

**Lead** : Prospect ou opportunité commerciale dans le CRM

**Locataire** : Personne qui loue un appartement

**Loyer Mensuel** : Montant du loyer payé chaque mois pour un appartement

## M

**Marge** : Pourcentage de marge appliqué sur le prix de base d'un voyage

**Merise** : Méthode de modélisation de données (MCD, MLD, MPD)

**MLD** : Modèle Logique de Données (Merise)

**MCD** : Modèle Conceptuel de Données (Merise)

**MySQL** : Système de gestion de base de données relationnelle

## O

**Odoo** : ERP open-source avec modules CRM, facturation, comptabilité

**ORM** : Object-Relational Mapping (mapping objet-relationnel, ex: SQLAlchemy)

## P

**ParentContact** : Contact d'un parent d'élève participant à un voyage scolaire

**Périmètre** : Limites fonctionnelles du système

**Permission** : Droit d'accès à une ressource spécifique

**Planning** : Programme détaillé d'un voyage avec activités et horaires

**ProgramTemplate** : Modèle de programme préconstruit pour une destination

**Professeur** : Enseignant qui organise un voyage scolaire

## Q

**Quote** : Devis (en anglais)

**QUOTE_SENT** : Statut d'un voyage lorsque le devis a été envoyé

**QUOTE_VALIDATED** : Statut d'un voyage lorsque le devis a été validé

## R

**RentalContract** : Contrat de location d'appartement

**REST** : Architectural style pour les APIs web (Representational State Transfer)

**Rôle** : Groupe de permissions attribué à un utilisateur

## S

**Service** : Couche de logique métier dans l'architecture

**SQLAlchemy** : ORM Python pour les bases de données

**Stripe** : Plateforme de paiement en ligne

**Synchronisation** : Mise à jour bidirectionnelle des données entre systèmes

## T

**Teacher** : Professeur dans le système

**TOTP** : Time-based One-Time Password (mot de passe à usage unique basé sur le temps)

**TransportPrice** : Prix du transport pour une destination et une date donnée

**Travel** : Voyage (scolaire ou linguistique)

**TravelDestination** : Association entre un voyage et une destination (voyages multi-destinations)

**TVA** : Taxe sur la Valeur Ajoutée (20% en France)

## U

**UML** : Unified Modeling Language (langage de modélisation unifié)

**User** : Utilisateur du système avec compte et authentification

## V

**Vue.js** : Framework JavaScript pour créer des interfaces utilisateur

**Voyage Linguistique** : Voyage ouvert au public avec inscription et paiement en ligne

**Voyage Scolaire** : Voyage organisé par un professeur pour ses élèves

## W

**Workflow** : Processus métier séquentiel avec étapes et validations
